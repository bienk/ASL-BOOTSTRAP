A cosa serve bootstrap?

Bootstrap è un framework front-end gratuito per uno sviluppo web più rapido e semplice Bootstrap include modelli di progettazione basati su HTML e CSS per tipografia, moduli, pulsanti, tabelle, navigazione, modali, caroselli di immagini e molti altri, oltre a plug-in JavaScript facoltativi Bootstrap ti dà anche la possibilità di creare facilmente modelli reattivi

Cosa devi importare per creare una pagina web con bootstrap

Esistono due modi per iniziare a utilizzare Bootstrap 4 sul tuo sito web. Puoi: Includi Bootstrap 4 da un CDN Scarica Bootstrap 4 da getbootstrap.com

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

CDN (Content Delivery Network). MaxCDN fornisce supporto CDN per CSS e JavaScript di Bootstrap. Devi anche includere jQuery. Un vantaggio dell'utilizzo della CDN Bootstrap 4: molti utenti hanno già scaricato Bootstrap 4 da MaxCDN quando visitano un altro sito. Di conseguenza, verrà caricato dalla cache quando visitano il tuo sito, il che comporta tempi di caricamento più rapidi. Inoltre, la maggior parte dei CDN farà in modo che, una volta che un utente richiede un file da esso, verrà servito dal server più vicino a loro, il che comporta anche un tempo di caricamento più veloce.

A cosa serve questo attributo del tag meta? content="width=device-width, initial-scale=1

La width=device-widthparte imposta la larghezza della pagina in modo che segua la larghezza dello schermo del dispositivo (che varierà a seconda del dispositivo). La initial-scale=1parte imposta il livello di zoom iniziale quando la pagina viene caricata per la prima volta dal browser.

Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)

La .containerclasse fornisce un contenitore a larghezza fissa reattiva La .container-fluidclasse fornisce un contenitore a larghezza intera , che copre l'intera larghezza del viewport
